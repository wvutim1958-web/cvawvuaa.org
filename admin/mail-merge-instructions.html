1Install "Yet Another Mail Merge"

This is a FREE Gmail add-on that lets you send personalized mass emails.

  1. Go to Google Workspace Marketplace
  2. Search for "Yet Another Mail Merge" (YAMM)
  3. Click Install
  4. Authorize it with cvcwvuaa@gmail.com
  5. Grant the requested permissions
Install YAMM

2Export the Mail Merge CSV

From the Outstanding Dues Report, click the red "‚úâÔ∏è Gmail Mail Merge" button.

This downloads a CSV file with columns:

  • Email - Member's email address
  • FirstName - First name only
  • LastName - Last name
  • FullName - Complete name
  • MembershipType - Individual or Family
  • AmountDue - Dollar amount owed
  • InvoiceURL - Direct link to their invoice

3Import to Google Sheets

  1. Open Google Sheets
  2. Create a new blank spreadsheet
  3. Click File ‚Üí Import
  4. Click Upload tab
  5. Select the mail merge CSV you just downloaded
  6. Choose "Replace spreadsheet"
  7. Click Import data
