📚 Quick Start Guide

1. Member Database: View all 134 members, edit details, add/delete members

2. Payment Tracking: See all payments from all members, filter by type (dues/donations)

3. Outstanding Dues: Find who hasn't paid → generate invoices → export email lists

4. Financial Ledger: Track all income/expenses, reconcile bank statements

📄 How to Generate Invoices & Receipts:
Invoices (for unpaid members):
→ Go to Outstanding Dues Report → Click "📄 Invoice" button next to member name

Receipts (for paid members):
→ Go to Member Database → View member details → Click "📄 Receipt" button on any payment

💡 Tip: All data is stored in Firebase. Click "Member Database" above to access your 134 members!
